## 📚 What's Covered

### 🐍 Python Core
- Data Types — int, float, string, bool
- Lists, Tuples, Sets, Dictionaries
- Conditional Statements (`if`, `elif`, `else`)
- Loops — `for`, `while`, nested loops
- Functions — definition, arguments, return values
- Lambda Functions
- List & Dictionary Comprehensions
- File Handling
- Exception Handling

### 🔢 NumPy
- Arrays — 1D, 2D, 3D
- Array Operations & Broadcasting
- Indexing & Slicing
- Mathematical & Statistical Functions
- Reshape, Flatten, Transpose

### 🐼 Pandas
- Series & DataFrame creation
- Reading CSV / Excel files
- Data Selection — `loc`, `iloc`
- Filtering & Sorting
- Handling Missing Values
- GroupBy & Aggregation
- Merge, Join, Concat

### 📊 Matplotlib
- Line, Bar, Scatter, Histogram plots
- Customization — titles, labels, colors, legends
- Subplots

- Distplot, Boxplot, Heatmap, Pairplot
- Count Plot, Violin Plot
- Styling & Themes

### 📐 SciPy
- Statistical tests
- Hypothesis Testing basics
